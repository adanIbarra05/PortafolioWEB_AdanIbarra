# Proyecto 4 — REST API Dog API

## 1. Nombre del proyecto

REST API Dog API

## 2. Objetivo del proyecto

Consumir una API REST utilizando PHP y cURL para obtener información e imágenes de diferentes razas de perros, permitiendo la integración de servicios web externos dentro de una aplicación web.

## 3. Problema que resuelve

Permite consultar información dinámica proveniente de un servicio externo, facilitando el acceso y visualización de datos actualizados sobre razas de perros mediante una interfaz web sencilla.

## 4. Tecnologías utilizadas

- PHP
- REST API
- cURL
- JSON
- XAMPP
- HTML

## 5. Conceptos aplicados

- APIs REST
- Solicitudes HTTP GET
- cURL en PHP
- JSON
- Consumo de servicios web
- Integración de APIs externas
- Desarrollo web con PHP


## 6. Instrucciones de ejecución

1. Registrarse en The Dog API.
2. Obtener la API Key.
3. Instalar XAMPP.
4. Activar Apache desde XAMPP Control Panel.
5. Habilitar la extensión cURL en el archivo `php.ini`.
6. Crear la carpeta del proyecto dentro de:

```bash
C:\xampp\htdocs\dogapi
```

7. Copiar los archivos PHP del proyecto.
8. Abrir el navegador.
9. Ejecutar:

```bash
localhost/dogapi/buscador.php
```

10. Seleccionar una raza desde la lista desplegable.
11. Presionar buscar.
12. Visualizar las imágenes obtenidas desde la API.

## 7. Reflexión personal

### ¿Qué aprendí?

Aprendí a consumir APIs REST utilizando PHP y cURL, realizar solicitudes HTTP, procesar respuestas JSON y conectar una aplicación web con servicios externos.

### ¿Qué fue difícil?

La parte más complicada fue configurar correctamente cURL, comprender el funcionamiento de las APIs REST y manejar la autenticación mediante la API Key.

### ¿Qué mejoraría?

Mejoraría la interfaz visual del sistema, agregaría más información de cada raza, optimizaría la búsqueda y añadiría un diseño más moderno y amigable para el usuario.
