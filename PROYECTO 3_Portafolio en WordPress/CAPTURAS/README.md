# Proyecto 2 — Portafolio en WordPress

## 1. Nombre del proyecto

Portafolio en WordPress

## 2. Objetivo del proyecto

Crear un sitio web tipo portafolio utilizando WordPress para administrar contenido, configurar páginas, menús, temas y plugins.

## 3. Problema que resuelve

Facilita la creación y administración de sitios web profesionales sin desarrollar completamente desde código.

## 4. Tecnologías utilizadas

- WordPress
- Astra Theme
- Contact Form 7
- HTML
- CSS
- CMS

## 5. Conceptos aplicados

- CMS (Sistema de Gestión de Contenido)
- Configuración de páginas
- Menús de navegación
- Temas y personalización
- Plugins
- Formularios de contacto
- Diseño web


## 6. Instrucciones de ejecución

1. Instalar WordPress.
2. Configurar servidor local o hosting.
3. Acceder al panel administrativo.
4. Instalar el tema Astra.
5. Crear páginas principales.
6. Configurar menú y diseño.
7. Instalar Contact Form 7.
8. Visualizar el sitio terminado.

## 7. Reflexión personal

### ¿Qué aprendí?

Aprendí a utilizar WordPress, administrar contenido, instalar temas, configurar plugins y personalizar un sitio web.

### ¿Qué fue difícil?

Tuve dificultades con la configuración del menú, algunos elementos visuales y la integración del formulario.

### ¿Qué mejoraría?

Mejoraría el diseño visual, agregaría más contenido multimedia y optimizaría la experiencia del usuario.
