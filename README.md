## Descripción General

Este portafolio reúne las prácticas, proyectos y actividades desarrolladas durante la materia de **Desarrollo de Aplicaciones Web**, mostrando la aplicación de conocimientos relacionados con programación web, diseño de interfaces, consumo de APIs, administración de contenido y desarrollo de sistemas.

Dentro de este portafolio se incluyen proyectos orientados al desarrollo de páginas web, integración de bases de datos, uso de plataformas CMS y consumo de servicios web externos.
