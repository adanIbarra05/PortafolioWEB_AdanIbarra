# Proyecto 1 — Mini aplicación web

## 1. Nombre del proyecto

Mini Aplicación Web

## 2. Objetivo del proyecto

Desarrollar una página web básica utilizando HTML y CSS para comprender la estructura de un documento web, la organización del contenido y la aplicación de estilos visuales.

## 3. Problema que resuelve

Permite comprender los fundamentos del desarrollo web mediante la creación de una página sencilla, facilitando el aprendizaje inicial sobre estructura, diseño y visualización en un navegador.

## 4. Tecnologías utilizadas

- HTML
- CSS
- Navegador Web
- Visual Studio Code

## 5. Conceptos aplicados

- Estructura básica HTML5
- Etiquetas HTML
- Encabezados y párrafos
- Hojas de estilo CSS
- Diseño visual web
- Organización del contenido


## 6. Instrucciones de ejecución

1. Descargar o clonar el proyecto.
2. Abrir la carpeta del proyecto.
3. Abrir el archivo `index.html`.
4. Ejecutar el archivo desde cualquier navegador.
5. Verificar el funcionamiento del diseño y contenido.

## 7. Reflexión personal

### ¿Qué aprendí?

Aprendí la estructura básica de una página web utilizando HTML y CSS, así como la relación entre el código y su visualización en el navegador.

### ¿Qué fue difícil?

La parte más complicada fue comprender la organización correcta de etiquetas HTML y aplicar estilos CSS correctamente.

### ¿Qué mejoraría?

Mejoraría el diseño visual agregando más estilos, imágenes, botones y una estructura más moderna.
