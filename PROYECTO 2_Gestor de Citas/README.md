# Proyecto 2 — Sistema de Gestión de Citas

## 1. Nombre del proyecto

Sistema de Gestión de Citas

## 2. Objetivo del proyecto

Desarrollar un sistema web para registrar usuarios, iniciar sesión y administrar citas utilizando PHP y MySQL.

## 3. Problema que resuelve

Permite gestionar citas de manera organizada mediante un sistema web con autenticación de usuarios y operaciones CRUD para administrar la información.

## 4. Tecnologías utilizadas

- PHP
- MySQL
- PDO
- HTML
- CSS
- XAMPP
- phpMyAdmin

## 5. Conceptos aplicados

- CRUD
- MVC
- Conexión a base de datos
- Consultas preparadas
- Seguridad web
- Manejo de sesiones
- Autenticación
- Bases de datos relacionales


## 6. Instrucciones de ejecución

1. Instalar XAMPP.
2. Activar Apache y MySQL.
3. Abrir phpMyAdmin.
4. Importar la base de datos del proyecto.
5. Copiar la carpeta del proyecto dentro de:

```bash
C:\xampp\htdocs\sistema_citas
```

6. Abrir el navegador.
7. Ejecutar:

```bash
localhost/sistema_citas
```

8. Registrar un usuario nuevo.
9. Iniciar sesión.
10. Crear, visualizar, editar y eliminar citas.

## 7. Reflexión personal

### ¿Qué aprendí?

Aprendí a desarrollar una aplicación web dinámica utilizando PHP y MySQL, realizar conexiones seguras a bases de datos, implementar autenticación de usuarios y aplicar operaciones CRUD.

### ¿Qué fue difícil?

Las partes más complicadas fueron la conexión entre PHP y MySQL, el manejo de sesiones, la organización del proyecto mediante MVC y la implementación de consultas seguras.

### ¿Qué mejoraría?

Mejoraría el diseño visual del sistema, agregaría validaciones más completas, optimizaría la experiencia del usuario y añadiría funciones adicionales como búsqueda, filtros y edición avanzada de citas.
