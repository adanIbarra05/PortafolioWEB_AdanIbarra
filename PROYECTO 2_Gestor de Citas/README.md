# Proyecto 3 — Sistema de Gestión de Citas

## 1. Nombre del proyecto

Sistema de Gestión de Citas

## 2. Objetivo del proyecto

Desarrollar un sistema web para registrar usuarios, iniciar sesión y administrar citas utilizando PHP y MySQL.

## 3. Problema que resuelve

Permite gestionar citas de manera organizada mediante un sistema web con autenticación y operaciones CRUD.

## 4. Tecnologías utilizadas

- PHP
- MySQL
- PDO
- HTML
- CSS
- XAMPP
- phpMyAdmin

## 5. Conceptos aplicados

- CRUD
- MVC
- Conexión a base de datos
- Consultas preparadas
- Seguridad web
- Manejo de sesiones
- Autenticación


## 6. Instrucciones de ejecución

1. Instalar XAMPP.
2. Activar Apache y MySQL.
3. Importar la base de datos en phpMyAdmin.
4. Copiar el proyecto en `htdocs`.
5. Abrir:

```bash
localhost/sistema_citas
